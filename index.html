How to use CodeShare

If you are able to, we recommend using FileZilla or some other client to login to codeshare anonymously via ftp. And if you are downloading on a linux machine just find the link of the code you need, and then use wget https://sceneinternetservices.gq:1920/projects/yourusername/projectname/file.whatever.
